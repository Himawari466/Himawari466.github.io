Lebih produktif dengan tmux

Kenapa tmux?

  • Mudah digunakan.
  • Mudah dikonfigurasi.
  • Bisa synergy dengan vim / neovim.
  • Rata-rata ada di package manager banyak distro.
  • Bisa multitask di sesi terminal, tanpa gui, seperti headless server.
  • Mendukung attach dan detach session, sehingga lebih fleksibel.


Instalasi tmux

Instalasi tmux di Arch Linux

Lakukan update repo terlebih dahulu, saya asumsikan teman-teman menggunakan sudo untuk eskalasi hak akses super user / root:

% sudo pacman -Syy

Mulai instal tmux:

% sudo pacman -S tmux


Instalasi tmux di Ubuntu / Debian based

Lakukan update repo terlebih dahulu, saya asumsikan teman-teman menggunakan sudo untuk eskalasi hak akses super user / root:

$ sudo apt update

Mulai instal tmux:

$ sudo apt install tmux


Konfirmasi keberadaan tmux di sistem

Gunakan perintah which untuk mengecek keberadaan tmux:

~ % which tmux
/usr/bin/tmux

~ % tmux -V
tmux 3.1c


Instalasi xsel

xsel digunakan untuk sharing clipboard dari buffer tmux ke buffer system clipboard Xorg server (GUI).

Instalasi xsel di Arch Linux
~ % sudo pacman -S xsel


Instalasi xsel di Ubuntu / Debian based
$ sudo apt install xsel


Konfirmasi keberadaan xsel di sistem

Gunakan perintah which untuk mengecek keberadaan xsel:

~ % which xsel
/usr/bin/xsel


Membuat file konfigurasi ~/.tmux.conf

Membuat file konfigurasi ~/.tmux.conf:

$ touch ~/.tmux.conf

Edit file ~/.tmux.conf dengan text editor favorit teman-teman.
Berikut file konfigurasi tmux yang saya gunakan:

# remap modifier key dari Ctrl+b ke Ctrl+a, agar lebih mudah & praktis
unbind C-b
set -g prefix C-a
bind C-a send-prefix




# set peringatan notifikasi di status bar tmux jika ada sesi terminal yang urgent / berubah
setw -g monitor-activity on
set -g visual-activity on

# set default shell di tmux, saya menggunakan zsh
# ganti ke /usr/bin/bash jika menggunakan bash
#
# $ echo 0
#
# untuk cek
set -g default-shell /usr/bin/zsh

# set window mulai dari window 1, daripada window 0
set -g base-index 1

# set pane mulai dari pane 1, daripada pane 0
setw -g pane-base-index 1

# set pane, geometry child pane
set-window-option -g other-pane-height 25
set-window-option -g other-pane-width 80

# set pane, timeout highlight pane
set-window-option -g display-panes-time 1500

# set history limit di tmux
set -g history-limit 10000

# set timeout modifier key Ctrl+a
set -sg escape-time 10
set-option -g repeat-time 0

# mengaktifkan dukungan mouse
set-option -g mouse on

# reload config tanpa keluar / membuat sesi baru tmux
bind-key R source-file ~/.tmux.conf \; display-message "Konfig diperbaharui..."

#  mengaktifkan dukungan 256color
set -g default-terminal "xterm-256color"
set -ga terminal-overrides ",xterm-256color:Tc"

# mengaktifkan vi-mode / vi-binding di tmux
setw -g mode-keys vi
set -g mode-keys vi
set -g status-keys vi




# settingan keybind
bind-key C-x rotate-window
bind-key C-'\' detach
bind-key : command-prompt
bind-key a last-pane
bind-key c new-window
bind-key L clear-history
bind-key r refresh-client

bind-key q display-panes

bind-key + select-layout main-horizontal
bind-key = select-layout main-vertical
bind-key [ copy-mode
bind-key ] paste-buffer

bind-key enter next-layout
bind-key bspace previous-window
bind-key space next-window

bind-key v split-window -h
bind-key s split-window -v

bind-key -n M-S-h select-pane -L
bind-key -n M-S-j select-pane -D
bind-key -n M-S-k select-pane -U
bind-key -n M-S-l select-pane -R

# praktis, Ctrl+a dua kali untuk kembali ke window sebelumnya
bind-key C-a last-window

bind-key -T copy-mode-vi v send-keys -X begin-selection
bind-key -T copy-mode-vi y send-keys -X "copy-selection && run \"tmux save-buffer | xsel -i --clipboard\""
bind-key -T copy-mode-vi r send-keys -X rectangle-toggle

# [ untuk masuk scroll mode, tekan v untuk melalukan selection mode, tekan enter otomatis copy ke clipboard xsel
bind-key -T copy-mode-vi Enter send-keys -X copy-pipe-and-cancel "xsel -i --clipboard"




# set colorsheme tmux
set-option -g status-bg colour16
set-option -g status-fg colour255
set-window-option -g window-status-current-style bg=colour255
set-window-option -g window-status-current-style fg=colour11
set-window-option -g window-status-style bg=colour255
set-window-option -g window-status-style fg=colour9




# set status bar diatas
set-option -g status-position top

# set font status bar tebal (di terminal emulator)
set -g status-style bold

# set status bar refresh rate 1 detik
set-option -g status-interval 1

# set status bar bagian kiri untuk menampilkan suhu CPU
# bebas mau diisi apa, suka-suka
set-option -g status-left ' #(cat /sys/class/thermal/thermal_zone0/temp | cut -c -2)C | '

# set status bar bagian kanan untuk menampilkan kalender
# bebas juga, up to you
set-option -g status-right '| #[fg=colour4]%A#[fg=colour255], %d %B %Y [#[fg=colour11]%H:%M:%S#[fg=colour255]] '
