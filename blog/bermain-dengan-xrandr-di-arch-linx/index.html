Jadi display laptop saya rasanya kurang besar, kepikiran buat upscale layar / panning.

Terlihat di perangkat LVDS layar saya berada, dengan resolusi layar 1280x800.


Upscale layar 1.5 kali resolusi

Saya coba upscale resolusi layar 1.5 kali:

% xrandr --output "LVDS" --primary --pos "0x0" --scale 1.5x1.5

% xrandr
Screen 0: minimum 320 x 200, current 1920 x 1600, maximum 8192 x 8192
DVI-0 disconnected (normal left inverted right x axis y axis)
LVDS connected primary 1920x1200+0+0 (normal left inverted right x axis y axis) 331mm x 207mm
   1280x800      60.00*+  50.00  
   1280x720      59.97  
   1152x768      59.95  
   1024x768      59.95  
   800x600       59.96  
   848x480       59.94  
   720x480       59.94  
   640x480       59.94  
DisplayPort-0 disconnected (normal left inverted right x axis y axis)
VGA-0 disconnected (normal left inverted right x axis y axis)

Hasil upscale resolusi layar laptop saya dari 1280x800 menjadi 1920x1200 wkk!
Di layar fisik hasilnya buram tentu saja karena hardware layar saya tidak mendukung resolusi sebesar itu wkkk!
Option lain yang bisa digunakan di xrandr adalah:

--filter bilinear # (default)
# dan
--filter nearest

Mengembalikan ke semula upscale 1.5 ke scale 1.0:

% xrandr --output "LVDS" --primary --pos "0x0" --scale 1x1


Mencoba memperbesar resolusi layar dengan panning

Selanjutnya saya mau coba meninggikan / menambah resolusi layar saya ke 2K dengan panning.
Pertama saya pergi dulu ke sini aspect ratio calculator, saya kalkulasi dulu layar secara width & heigth (panjang & lebar layar) karena saya ingin mempertahankan aspect ratio layar.
Sesuai layar saya:
W1 = 1280
H1 = 800
W2 = 2560
otomatis H2 akan berisi 1600

Dari hasil kalkulasi resolusi layar dari awal 1280x800 menjadi 2560x1600 saya terapkan dengan xrandr:

% xrandr --output "LVDS" --mode 1280x800 --panning 2560x1600 --scale 1.0x1.0

% xrandr
Screen 0: minimum 320 x 200, current 2560 x 1600, maximum 8192 x 8192
DVI-0 disconnected (normal left inverted right x axis y axis)
LVDS connected primary 2560x1600+0+0 (normal left inverted right x axis y axis) 331mm x 207mm panning 2560x1600+0+0
   1280x800      60.00*+  50.00  
   1280x720      59.97  
   1152x768      59.95  
   1024x768      59.95  
   800x600       59.96  
   848x480       59.94  
   720x480       59.94  
   640x480       59.94  
DisplayPort-0 disconnected (normal left inverted right x axis y axis)
VGA-0 disconnected (normal left inverted right x axis y axis)

Sekarang saya legit punya layar resolusi 2K di layar laptop wkk…
Bisa geser - geser mouse ke pojok layar (panning) untuk scroll layar wkk!

Mengembalikan resolusi 2560x1600 ke semula 1280x800:

% xrandr --output "LVDS" --mode 1280x800 --panning 1280x800 --scale 1.0x1.0


Penutup

Use case / penggunaan misal kita ingin screenshot layar secara HD / detail tapi resolusi layar kita kecil sehingga menyebabkan blur pada screenshoter, nah bisa kita akali dengan cara ini.
Atau mungkin sedang bekerja dengan layar resolusi kecil di laptop netbook beresolusi 1024x768 tetapi ingin merubah resolusi menjadi besar.
