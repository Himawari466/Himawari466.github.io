Menginstal font TTF di Linux

Halo!
Selamat pagi teman-teman semua, di post keempat saya ini saya akan membagikan cara yang saya gunakan untuk menginstal font di GNU/Linux.

Gunakan daftar isi diatas & shortcut CTRL+F untuk memudahkan pencarian konten.

Persiapan

Asumsi saya sudah terinstal fontconfig, jadi tinggal instal font saja.
Di banyak distro fontconfig sudah terinstal by default satu paket dengan Xorg server.
Jika belum terinstal teman-teman bisa ke refer distro masing-masing untuk menginstal fontconfig.

Lokasi instal font secara system-wide:

/usr/share/fonts

Lokasi instal font secara local:

~/.local/share/fonts
~/.fonts/

Kita pakai yang local ~/.fonts/ saja karena mudah & untuk user yang sekarang digunakan saja.
Pertama buat directory untuk menyimpan font terlebih dahulu:

% mkdir ~/.fonts


Menyiapkan file font TTF

Saya asumsikan teman-teman sudah menyiapkan file font True Type Format (TTF).
Saya disini akan menggunakan font mononoki Nerd Fonts karena mudah dibaca,
yang di download dari https://github.com/ryanoasis/nerd-fonts/tree/master/patched-fonts,
dimana font disana sudah di patch dengan berbagai glyph / simbol.

Hasil download saya:

  • mononoki Bold Italic Nerd Font Complete Mono.ttf
  • mononoki Bold Italic Nerd Font Complete.ttf
  • mononoki Bold Nerd Font Complete Mono.ttf
  • mononoki Bold Nerd Font Complete.ttf
  • mononoki Italic Nerd Font Complete Mono.ttf
  • mononoki Italic Nerd Font Complete.ttf
  • mononoki-Regular Nerd Font Complete Mono.ttf
  • mononoki-Regular Nerd Font Complete.ttf


Instalasi font TTF

Membuat directory untuk menaruh font TTF:

% mkdir ~/.fonts/ttf

Lalu taruh semua file font TTF yang sudah disiapkan ke directory ~/.fonts/ttf.
Selanjutnya rebuild font dengan fontconfig:

% fc-cache -fv

Jalankan perintah tersebut setiap kali selesai menaruh font baru di ~/.fonts/ttf.


Mengecek font TTF yang terinstal

Saya sudah menginstal font, sudah rebuild font, sekarang saya cek font yang saya instal.
Selanjutnya saya gunakan perintah berikut:

% fc-list | grep 'mononoki'
/home/selene/.fonts/ttf/mononoki-Regular Nerd Font Complete Mono.ttf: mononoki Nerd Font Mono:style=Regular
/home/selene/.fonts/ttf/mononoki Bold Italic Nerd Font Complete.ttf: mononoki Nerd Font:style=BoldItalic
/home/selene/.fonts/ttf/mononoki Bold Nerd Font Complete.ttf: mononoki Nerd Font:style=Bold
/home/selene/.fonts/ttf/mononoki Italic Nerd Font Complete Mono.ttf: mononoki Nerd Font Mono:style=Italic
/home/selene/.fonts/ttf/mononoki Bold Italic Nerd Font Complete Mono.ttf: mononoki Nerd Font Mono:style=BoldItalic
/home/selene/.fonts/ttf/mononoki Bold Nerd Font Complete Mono.ttf: mononoki Nerd Font Mono:style=Bold
/home/selene/.fonts/ttf/mononoki Italic Nerd Font Complete.ttf: mononoki Nerd Font:style=Italic
/home/selene/.fonts/ttf/mononoki-Regular Nerd Font Complete.ttf: mononoki Nerd Font:style=Regular

Teman-teman bisa pipe dengan less, untuk mencari & mengecek font yang sudah terinstal:

% fc-list | less

Dari hasil pengecekan, font saya berhasil terinstal & siap digunakan.



Berikut ketika saya menggunakan font yang diinstal tadi.


Penutup

Demikian tutorial cara mudah instal font TTF di GNU/Linux, semoga isi post ini bisa bermanfaat buat teman-teman semua.
